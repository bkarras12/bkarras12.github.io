College Football Prediction Model

Project Overview

This project uses historical college football data to predict game outcomes. The models were built using data analytics techniques including feature engineering, regression modeling, and model evaluation.

  • Data sourced from CollegeFootballData API
  • Team-level efficiency and rating metrics
  • Linear and logistic regression models
