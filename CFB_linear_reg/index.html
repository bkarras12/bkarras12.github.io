CFB Prediction Model

Historical Analysis • Regression Modeling • Game Outcomes

Project Overview

This project leverages historical college football data to predict game outcomes with high fidelity. The models utilize advanced data analytics techniques including feature engineering, regression modeling, and rigorous evaluation.

  • Data Source: CollegeFootballData API
  • Key Metrics: EPA (Offense/Defense), PPA per play, ELO ratings, Season Stats, Talent Composite.
  • Methodology: Linear regression for margin of victory; Logistic regression for spread projections.
  • Performance: Averaging 73% Accuracy week-to-week.

Linear Regression Logic

The model predicts the margin of victory using a weighted coefficient formula. It isolates the difference in team quality and adjusts for home-field advantage.

ŷ = β₀ + β₁(Elo_diff) + β₂(Off_PPA_diff) + β₃(Def_PPA_diff) + β₄(Home_Field)

Variable Definitions:

  • ŷ : Predicted margin of victory
  • β₀ : Intercept value
  • Elo_diff : Delta in team Elo ratings
  • Off_PPA_diff : Offensive Predicted Points Added difference
  • Def_PPA_diff : Defensive Predicted Points Added difference
  • Home_Field : Binary indicator (0 or 1)
