College Football Prediction Model

Project Overview

This project uses historical college football data to predict game outcomes. The models were built using data analytics techniques including feature engineering, regression modeling, and model evaluation.

  • Data sourced from CollegeFootballData API
  • Taught the model on metrics such as EPA on offense and defense, PPA per play type, ELO ratings, Season player and team stats, and talent composit ratings
  • Linear and logistic regression models

Linear Regression Model

The linear regression model predicts margin of victory using the following equation:

ŷ = β₀ + β₁(Elo_diff) + β₂(Off_PPA_diff) + β₃(Def_PPA_diff) + β₄(Home_Field)

Where:

  • ŷ = predicted margin of victory
  • β₀ = intercept
  • Elo_diff = difference in team Elo ratings
  • Off_PPA_diff = offensive PPA difference
  • Def_PPA_diff = defensive PPA difference
  • Home_Field = home field indicator (0 or 1)
