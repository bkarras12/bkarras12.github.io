# Portfolio Redesign — Bold Rebrand with Collapsible Sections

## Overview
Redesign bkarras12.github.io portfolio page with a bold new color palette, collapsible project sections, glassmorphism card styling, and reorganized project categorization.

## Layout & Structure
- Header: Name + refined About Me dropdown
- 4 collapsible sections stacked vertically, all open by default
- Each section: full-width clickable header bar with title (left) and rotating chevron (right)
- Gradient accent line under each section header
- Cards inside each section: CSS grid (auto-fit, minmax 280px, 1fr)
- Smooth collapse/expand via max-height + opacity transition
- Footer: cleaned up, subtle

## Visual Design — Bold Rebrand
- **Background:** Deep dark base (#0a0a0f) with colored radial gradient blurs (teal/cyan + purple/violet)
- **Primary accent:** Cyan/teal (#00d4aa)
- **Secondary accent:** Violet/purple (#7c5cfc)
- **Section headers:** Semi-transparent backdrop, left-side gradient border accent, each section a slightly different hue
- **Cards:** Glassmorphism — semi-transparent bg with subtle backdrop-blur, border glow on hover matching section color
- **Hover states:** Cards lift + border glow intensifies + box-shadow bloom
- **Typography:** Tighter letter-spacing on headers, larger section titles, high-contrast tags
- **Section dividers:** Gradient lines (cyan to purple fade)
- **Tags:** Pill-shaped, semi-transparent with accent border

## Collapsible Sections — Interaction
- Entire section header bar is clickable toggle
- Chevron icon on right rotates 180deg on collapse
- Smooth animation (max-height + opacity)
- All sections open by default on load
